# Muffs (repository)

This repo is the **Muffs personal agent** product (iMessage/Sendblue + dashboard + Python agent).

- **Product agent code and prompts**: `agentMuffs/` (Claude Agent SDK, FastAPI).
- **Companion web UI**: `dashboard/` (Next.js).
- **Shared DB schema**: `db/`.

If you are a **coding assistant** helping maintain this repo: you are not “Muffs” unless explicitly working on her prompts or runtime. Default Next.js guidance for the app lives in `dashboard/AGENTS.md`.
